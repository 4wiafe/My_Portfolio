# Richmond Wiafe

## Overview

This is my personal portfolio website, a single-page, fully responsive frontend built with pure HTML, CSS, and JavaScript. It is designed to showcase my projects, skills, and development thinking in a clean and professional way.
The core philosophy behind how I build is:

Interfaces should not just look good — they should guide users to take action.

Every project I build is grounded in a real problem and shaped by how users think, behave, and make decisions.

## Features

### 1. Hero Section

Displays my name, role, and headline.
Includes two CTA buttons: View My Work and Get In Touch.
Animated background blobs and a grid texture for visual depth.
Floating badges highlight key skills.

### 2. Tagline Strip

An infinite auto-scrolling ticker between the hero and about sections.
Pauses on hover.
Highlights core skills and areas of focus.

### 3. About Section

Tells my development story and philosophy.
Highlights three key UX questions I think about when building:

What drives users to take action?
Where do they drop off, and why?
How can the experience be clearer, faster, and more effective?

### 4. Projects Section

Features a detailed card for each project.
Each card includes what was built, the impact and thinking behind it, the tech stack, and links to the live site and source code.

### 5. Skills Section

Five skill category cards covering:

Front-End Development
Core Concepts
Development Approach
What I Focus On
Tools and Workflow

### 6. Contact Section

Centred layout with a clear headline and supporting copy.
Links to email, GitHub, and LinkedIn.

### 7. Footer

Brand description and social icon links.
Navigation links and project links for quick access.
